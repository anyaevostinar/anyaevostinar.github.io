## Overview
The final topic that we'll discuss this term is a small one (kidding): the Internet. You know (sort of) how computers can send information to each other, but how do they find each other in the first place?

## Basic Learning Objectives
Before class, you should be able to:
* Define resolver and ISP in this context
* Define the root server and TLD in this context

## Advanced Learning Objectives
After class, you should be able to:
* Draw out the connections between the different entities involved with DNS
* Explain how redundancy is built in to the system
