## Color blindness
[Color blindness simulator for images](http://www.color-blindness.com/coblis-color-blindness-simulator/): upload or drag-and-drop an image and choose from several filters.

[Color blind webpage filter](https://www.toptal.com/designers/colorfilter): enter a URL and compare side-by-side how people with normal vision and people with various forms of color blindness see the same page.

[Daltonization](http://www.vischeck.com/daltonize/): an algorithm to correct for colors that are hard to detect by red-green color blind people. Note that the "try it" functionality no longer seems to be working.

[Color safe palette generator](http://colorsafe.co/).
